- SOBRE A PASTA 'midias/imagens'

Essa pasta serve para salvar arquivos de imagens utilizados pelos módulos 
do sistema, não pelo sistema necessariamente, e nem para os aplicativos administrados pelo
nano admin. Para usar arquivos de mídia em aplicativos ou para o nano admin no geral, leia 
os arquivos 'README-pt-br.txt' contido neles.

Crie tantos arquivos e subpastas aqui quanto nescessário,
para manter seu projeto o organizado e compreensível.
