- SOBRE A PASTA 'assets/images/'

Essa pasta contem as imagens para o uso exclusivo para LAYOUT do sistema, e não
para o uso dos módulos. Para usar imagens administradas pelos módulos,
existe a pasta 'midias'.

Crie tantos arquivos e subpastas aqui quanto nescessário,
para manter seu projeto o organizado e compreensível.
